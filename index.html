NIEM is a community–driven, standards–based approach to exchanging information. This Open Source Spot is for the community to have access to resources that will help them use NIEM and make NIEM easier for others to use.

 NIEM–UML

The NIEM UML Profile allows UML tool developers to build NIEM into their products. Follow and contribute to the development of the NIEM UML Profile for v3.0.

Learn MoreGitHub Repo

 NIEM Modeling Tool

The NIEM Modeling Tool is an open source tool that simplifies building NIEM exchanges using NIEM–UML. Use and improve the community–owned tool here.

Relevant Projects and Links

NDR Test Suite
Access tools available for the testing and use of the NIEM 3.0 Naming and Design Rules (NDR) Schematron rules.

NIEM Developer Support Network (coming soon)
The NIEM developer network will serve as a one–stop–shop for IEPD templates, examples and tutorials for developers and implementers.

References and Specs
References and specifications assist the IEPD developers by providing them an understanding and rationale of the NIEM technical architecture. These documents include the Naming and Design Rules, Model Package Description (MPD) specification, among others.

Current Release
The NIEM model is in version 3.0. The current release contains the schemas, as well as additional formats such as Excel to be downloaded.
