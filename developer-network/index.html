--- title: NIEM Developer Network ---

Welcome! The NIEM developer support network offers guidance to those working to develop NIEM Information Exchange Package Documentation (IEPD) packages. This site is composed of IEPD examples, a step–by–step tutorial on how to build an IEPD, a reference list of normative rules from the NIEM Conformance Targets Attribute Specification, Model Package Description specification, and Naming and Design Rules, as well as a dictionary of common terms associated with NIEM.

NOTE: Some developer support network features are currently in development. Let us know if you have feedback or ideas for site enhancements.
